# Expense Tracker (Java)

## 📌 Project Description
This is a simple Expense Tracker application developed using Java. The purpose of this project is to help users record and manage their daily expenses. It allows users to add, view, and track their spending, making it easier to understand personal financial habits.

## 🚀 Features
- Add new expenses
- View all recorded expenses
- Track daily spending
- Simple console-based interface

## 🛠️ Technologies Used
- Java
- OOP Concepts

## ▶️ How to Run
1. Clone or download the repository
2. Open the project in any Java IDE (Eclipse / IntelliJ / VS Code)
3. Compile and run the Java file
4. Follow the instructions in the console

## 🎯 Purpose
This project was created to practice Core Java concepts such as classes, objects, and basic program logic.
